Japanese Vertical Text

When using writing modes to layout text, does it affect the way Grid layout works? I've prepared some Japanese Lorem Ipsum text and laid it out using the same grid layout I created for Ichi-go Ichi-e.

I need to do more research to see what looks normal for Japanese readers using vertical layouts as opposed to the RTL horizontal layout that I've seen the most.

What I didn't realize when I started working on this idea is that if the text flows right to left you should count the position of text for right to left as well. When defining the placement of an object in the grid the first value represents the right-side position of the content and the second one refer to the left-side ending position on the grid.

The next exploration in this area will be to mix vertical and horizontal Japanese content to see what the effect is. Is it the same as having western text mixed?
